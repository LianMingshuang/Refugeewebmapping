Web Mapping Data and Sources

We sourced GIS data that would support refugee project policy formulation, implementation and monitoring.

GIS Data Sources

  1. We sourced some GIS data from United Nations Office for the Coordination of Humanitarian Affairs (UN-OCHA). UN-OCHA Website
  2. We sourced other GIS data from Humanitarian Data (Humdata) open data website.
  3. We collected data on: sea ports, health facilities, educational facilities, and populated sites, boundaries. Except boundaries, all data is point data.

The following are the GIS layers vector data used in developing the web map:

Data Layer Data Description Data Source
Sea Ports This dataset shows all the sea ports in Venezuela. It's relevant to shows exit ports for refugee migration. Humanitarian Open Data
Health Facilities This dataset shows the list of operating health facilities. Attributes included: Name,Nature of Facility, Activities, Lat, Long Humanitarian Open Data
Educational Facilities This dataset shows the list of operating educational facilities. Attributes included: Name,Nature of Facility, Activities, Lat, Long Humanitarian Open Data
Country Boundaries Contains country political boundary definition and names from the United Nations Protocol and Liaison Office (DGACM) Humanitarian Open Data
IOM South America Offices Contains locations of IOM field ofices in South America. The data is available in GeoJSON format and can be downloaded from the Github repository at the link aside. IOM South America Field Offices
