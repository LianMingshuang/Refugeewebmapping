Web Design & Mapping Technologies

We combined several technologies to design the website and develop web mapping as follows

  • Web UI/Front End
  • Web Design Libraries
  • GIS Hosting Technologies
  • Web Mapping Technologies
Technology Advantages Disadvantages
  1. Hypertext Markup Language 5 (HTML5)
  2. Cascading Style Sheets (CSS)
  3. Responsive Web Design (RWD)
  4. JavaScript
  1. Allows semantic modern user interface design
  2. Enables centralization of web page styles
  3. Enables UI interface activity design

There are no disadvantages in using these technologies for web UI design.

They are the only such technology available for web UI design
Technology Advantages Disadvantages
  1. Boostrap 4
  2. jQuery
  3. Popper JS
  1. Allows grid / flex -based web UI design
  2. Enables responsive and mobile-first UI design
  3. Enables interactive/agile UI design

They require high learning curve to master

These technologies have to be used in combination to be effective
Technology Advantages Disadvantages
  1. GitHub Pages
  2. ESRI ArcGIS Online (AGOL)
  3. GitHub
  1. GitHub Pages allows hosting of websites
  2. AGOL allows free to low cost hosting of GIS data
  3. GitHub enables online hosting of code and application version control
  4. GitHub enables centralized sharing of code for team programming
  1. There is a high learning curve to master these technologies
  2. AGOL has low-level GIS hosting capabilities compared to ArcGIS Server, MapServer, GeoServer etc
  3. The free offerrings of these technologies may have security issues.
Technology Advantages Disadvantages
  1. ESRI JavaScript API
  2. GIS Web Service APIs
  1. ESRI JavaScript API allows rendering of GIS layers in web environments
  2. ESRI JavaScript API allows online visualization and geoprocessing of GIS layers
  3. GIS web APIs host and provide end points for online GIS data access
  4. ESRI JavaScript has better visualization and geoprocessing capabilities compared to other online mapping technologies i.e Leaflet JS and Mapbox GL
  1. There is higher learning curve to learn these technologies
